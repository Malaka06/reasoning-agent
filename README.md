# Reasoning Agent

## C’est quoi ce projet ?
Reasoning Agent est une application Streamlit.

On peut lui poser une question (data, business ou ambiguë),
et l’application montre :
- la réponse
- le raisonnement derrière la réponse
- les alternatives possibles

Ce projet montre **comment je réfléchis**, pas seulement ce que je réponds.

---

## Pourquoi j’ai créé ce projet
Dans les métiers de la data, on évalue souvent :
- la capacité de raisonnement
- la prise de décision
- la gestion de l’incertitude

Un CV ne montre pas cela.
Ce projet, si.

---

## Ce que ce projet n’est PAS
- Ce n’est pas un chatbot qui parle à ma place
- Ce n’est pas une automatisation de candidatures
- Ce n’est pas un CV déguisé

C’est une **simulation de ma façon de raisonner**.

---

## À quoi il sert
- Montrer mon raisonnement en entretien
- Discuter de problèmes data réels
- Expliquer ma manière de travailler

---

## Stack technique (vue d’ensemble)
- Python
